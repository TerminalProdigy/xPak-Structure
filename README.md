# xPak
#### xPak - Umbrella project. Incorporates everything.
#### xPak.Core - Core utilities, extensions, base classes, models, enums, interfaces, constants
#### xPak.Data - Database helpers
#### xPak.Integration - APIs, SDK wrappers, Adapters
#### xPak.Logging - Exception handling utilities
#### xPak.UI - Message boxes, dialogs, popups, Custom controls or components, UI helpers or services
